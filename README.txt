
BASIL JAIJU PORTFOLIO - README
==============================

Overview
--------
This is a personal portfolio website for Basil Jaiju, built using HTML and styled with an external CSS file (main.css).
The site showcases personal projects, provides a brief introduction, and includes contact options.

Website Structure
-----------------

1. Header
   - <div id="header">: Displays the portfolio owner's name "BASIL JAIJU".

2. Navigation Bar
   - <div id="navbar">: Contains navigation links:
     * HOME – index.html
     * ABOUT – aboutpage.html
     * PROJECTS – projects.html
     * CONTACT – Opens a contact popup

3. Intro Section
   - <section id="intro">: Displays an image and greeting message "------HELLO I'm Basil--------"

4. Works Preview
   - <div id="myself">: Short text introduction before project section

5. Latest Projects
   - <section id="latest_proj">: Displays latest work in a card layout:
     * MEME CORNER – memepage.html
     * RAILWAY TIMING – railwaytiming.html
     * Note on Nature – naturenote.html

6. Contact Popup
   - <div id="popup_div"> and <div id="myPopup">: Popup modal for contact details (email and phone)

7. Footer
   - <div id="footer_container">: Contains copyright
     and social media links (Instagram, GitHub)

Notes
-----
- External styles are defined in 'main.css'.
- Some images are sourced from URLs, others are local (like 'instalogo.png').
